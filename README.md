# Minimal Le QCM

## Prerequisites

- Copy `.env.example` to `.env` and fill in the values.

## Commands

Install dependencies:
```bash
pnpm install
```

Run the app:
```bash
pnpm dev
```

Docker compose
```bash
# Build
docker-compose -f docker-compose.prod?yml up
# Run
docker-compose -f docker-compose.prod.yml up
```
